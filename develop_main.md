1. ⚡️ Hero Section: "지금 가장 핫한 월드컵" (상단 배너)
현재는 바로 리스트(Popular WorldCups)가 나오는데, 시선을 확 사로잡는 Hero 배너가 필요합니다.

UI 구성: 화면 상단에 가장 인기 있는 월드컵 1개(또는 Top 3 캐러셀)를 크게 배치합니다.

디자인 디테일:

배경에 해당 월드컵의 1위 이미지나 대표 이미지를 꽉 채우고 backdrop-blur-sm (블러) 처리를 합니다.

"실시간 참여자 급상승 🔥" 같은 뱃지를 붙입니다.

[지금 바로 시작하기] 버튼을 아주 크고 화려한 그라데이션(bg-gradient-to-r from-violet-500 to-fuchsia-500)으로 배치합니다.

Why? 사용자의 고민 시간을 줄여주고 즉시 플레이를 유도합니다.

2. 🏷️ Category Chips: "취향대로 골라먹기"
스크린샷을 보면 모든 주제가 섞여 있습니다. 사용자가 관심사를 바로 찾을 수 있도록 필터가 필요합니다.

UI 구성: Popular WorldCups 타이틀 바로 아래에 **가로 스크롤(Horizontal Scroll)**이 가능한 칩(Chip) 버튼을 배치합니다.

항목: 🔥 전체, 💻 개발, 🍔 음식, 🎵 연예, 🤣 유머, ❤️ 연애

인터랙션: 선택된 칩은 형광색(border-fuchsia-500)으로 강조되고, 리스트가 필터링됩니다.

3. 🃏 Card UI Upgrade: "클릭하고 싶은 카드 만들기"
현재 카드는 정보 전달 위주입니다. 조금 더 게임(Game)스럽게 바꿔봅시다.

썸네일: 현재 32 Plays 뱃지는 좋습니다. 우측 상단에 Best나 New 리본을 추가해보세요.

타이틀: 폰트 두께를 font-extrabold로 키우고, 긴 제목은 말줄임표(...) 처리보다 2줄까지 보여주는 게 좋습니다.

Hover 효과 (PC):

마우스를 올리면 카드가 살짝 떠오르며(hover:-translate-y-1) 테두리에 네온 빛(shadow-[0_0_15px_rgba(139,92,246,0.5)])이 들어오게 합니다.

배경 이미지가 살짝 확대(scale-105)되는 효과를 줍니다.

Action 버튼 분리:

현재는 카드 전체가 링크인 것 같지만, 하단에 [▶ 플레이] 버튼과 [📊 랭킹 보기] 버튼을 아이콘으로 작게 분리하면 더 직관적입니다.
